System Info

The System Info option enables you to view the following aspects associated with the OptiProERP Configurator:

 

1. Version Number

2. User ID

3. Name of the Logged in company

4. Name of the Language

 

 

6.1 Procedure

 

1. On the top right hand side of the screen, click the icon.

2. On the resulting window, click the System Info icon.

3. The resulting System Info window displays the details such as: the Version Number, User ID, Name of the Logged in company, and the name of the Language.

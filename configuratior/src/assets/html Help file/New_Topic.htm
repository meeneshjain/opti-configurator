Home Page

On clicking the Login button, the Configurator Home page is displayed. The left pane on this page displays the following sub-menus:

 

• Home

• Item Code Generation

• Feature Model

• Feature BOM

• Feature BOM

• Model BOM

• Routing

• Rule Workbench

• Configuration Wizard

 

The top-right pane displays the user icon exists. Use the icon for system info and to log out of the application.

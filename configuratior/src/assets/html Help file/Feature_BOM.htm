Feature BOM

A feature BOM is list of intermediate components which forms a set of feature options available. For instance, a mobile (an item) can have various models (M10, M 20, M30), with varying features. Its feature BOM can include RAM: 2 GB, 3 GB, 4 GB and so on.

 

The feature screen enables you to:

 

• Create a Feature BOM entry

 

• Explode the BOM to ascertain the components

 

• Ascertain the associated BOMs

 

 

Procedure

 

1. On the Feature BOM screen, click the Add Row button.

 

2. On the resulting window, select the feature code by clicking the lookup available near the Feature Code field. On clicking this lookup, the resulting window displays all the existing Feature IDs along with their respective Code and Name.

 

3. The values in the Display Name and Feature Description fields are picked automatically from the Feature Master screen.

 

4. If you want to add any accessory for the feature, select the option as Yes available in the Accessory field. Otherwise select the option as No.

 

Note: If you have selected the option as Yes, you will be able to add the accessories with the help of the Model BOM screen.

 

5. If you do not want to add multiple default features, or items, or values select the Multi Select option as No.

 

6. The Min Selectable check box will be displayed. Click the drop down available near this check box and select any one of the following values:

 

  • If you select the value as 0, all the check boxes displayed in the Default column will change to buttons and will be disabled.

  • If you select the value as 1, all the check boxes displayed in the Default column will change to buttons. You can select any one of the checkboxes.        

                  

7. If you want to add multiple default features, or items, or values select the Multi Select option as Yes.

 

8. Enter the value in the Min Selectable field.

 

Note: You can also enter the value as 0 in this field.  

 

9. Enter the value in the Max Selectable field.

 

Note: The value entered in the Max Selectable field cannot be greater than the total number of rows added in the grid.

 

Example: If you have entered the value as 2 in the Max Selectable field, and added 1 row in the grid. On saving the record, you will receive a message stating that Maximum Selectable cannot be greater than total rows.  

 

10. Click the Add Row button available in the middle of the grid. On clicking this button, the following fields will be displayed.

 

11. Select the type by clicking the drop down menu available on the Type field. On clicking this drop down the following options will be displayed:

 

• Feature

• Item

• Value

 

12. The lookup available on the Feature/ Item/Value field displays the data based on the type of selection on the Type field.

 

• Type selected as Feature on the Type field: If you have selected the type as Feature in the Type field, the lookup will display all the existing Feature IDs. You need to select the Feature ID of your choice.

 

• Type selected as Item on the Type field: If you have selected the type as Item in the Type field, the lookup will display all the existing Items. You need to select the Item of your choice.

 

• Type selected as Value on the Type field: If you have selected the type as Value in the Type field, you need to enter the value in the Feature/ Item/Value field.

 

13. Enter a display name associated with the Feature/ Item/Value in the Display Name field. The Display Name enables easy identification of the same.

 

14. Enter the quantity in the Quantity field.

 

15. Select the check box available in the Default column if you want to make the Feature, Item, or Value as default.

 

16. Enter any special comments if you want to in the Remarks column.

 

17. The Propagate Quantity is applicable only for the type selected as Feature and Item in the Type field. Select the checkbox available in this field if you want to make them as the Propagate Quantity.

 

18. The Price Source field is applicable only for the type selected as Item in the Type field. On clicking the lookup ( ) available near this field, the resulting window displays all the existing items. You need to select an item of your choice.  

 

19. If you want to attach an image, click the Choose Image button available in the Attachment column. On the resulting window, browse, locate, and select the image. The image selected by you will be displayed in the Image column.

 

20. Click Save button save the record.

 

21. Alternatively, click Cancel button to close screen. The changes made by you will not be saved.

 

9.1 Resequencing the lines in the grid

 

On selecting any line in the grid, the colour of the line changes to pink colour. Further two arrow marks

upward, and downward are displayed in the top portion of the grid. Clicking the upward arrow moves the selected line to the top portion in the grid. Clicking the downward arrow moves the selected line to the bottom portion in the grid.

Key Terms

 

Feature Code: The lookup available near this field enables you to select the feature code for the product.

 

Display Name: The value in this field is picked automatically from the Feature Master screen.

 

Feature Description: The value in this field is picked automatically from the Feature Master screen.

 

Accessory: This field enables you to add an accessory for the feature.

 

Multi Select: This field enables you to add either a default feature, or item, or value.

 

Yes: If you select this option, the Max Selectable field will be enabled.

 

No: If you select this option, the Multi Select option will not be applied.

 

Min Selectable: This field enables you to enter the minimum selectable value.

 

Max Selectable: This field gets enabled on selecting the Multi Select option as Yes.

 

Add Row button: This button enables you to add rows in the grid.

 

Type: The dropdown menu available near this field enables you to select the type of your choice.

 

Feature / Item / Value field: The lookup available near this field enables you to select the data based on the type of selection on the Type field.

 

Display Name: This field enables you to enter a display name for the Feature/ Item/Value.

 

Quantity: This field enables you to enter the quantity.

 

Default: This column enables you to select either the Feature, or the Item, or the Value as the Default one. The Feature / Item / Value which you have made as default here will be

used on the Configuration Wizard screen.     

 

Remarks: This field enables you to enter any special comments.

 

Propagate Quantity: The propagate quantity refers to multiplication or increase in BOM item quantity with a subsequent increase in quantity of finished good item to be produced.

Selecting this check box proportionately increases the BOM item quantity while configuring the product and gets reflected in in the Model BOM to Configure Product within

Configuration Wizard screen.

 

Price Source: The Price Source field is applicable only for the type selected as Item in the Type field. On clicking the lookup available near this field, the resulting window

displays all the existing items. You need to select an item of your choice.

 

Attachment: This field enables you to include an image of your choice.

 

Image: If you have attached an image, this will be displayed in this column.

 

Save: Click this button to save the record.

 

Cancel: Clicking this button cancels the screen, and the record will not be saved.

 

9.2 Explode BOM

 

The Explode button available in the Feature BOM screen provides a Tree View of the Feature BOM. On clicking this button, the Parent and Child items associated with the Feature BOM will be displayed.

 

9.3 Associated BOM

 

The Associated BOM button available in the Feature BOM screen displays all the BOMs associated with the Feature BOM. On clicking this button, the resulting Associated BOM window displays all the BOMs associated with it.
